# CrackTrack - Self Improvement & Exam Tracker

A simple offline app for tracking:
- Daily study tasks
- Study hours and MCQs
- Mock test scores
- Habits
- Daily journal
- Exam countdown for Punjab Excise Inspector exam on 30 May 2026

## How to use
1. Extract the ZIP file.
2. Open `index.html` in Chrome or Edge.
3. Your data saves automatically in the browser/device.

## Optional: Install like an app
Open `index.html` in Chrome/Edge, then use:
Menu > Save and Share / Install page as app

## Backup
Use the Export Backup button inside the app to save your data as JSON.
