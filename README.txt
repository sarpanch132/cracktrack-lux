CrackTrack
==========

Premium Punjab Excise Inspector study tracker PWA.

What is included:
- Dashboard with exam countdown, progress ring, daily stats, timeline, tasks, and recommendations
- Editable daily schedule for running, study, and night revision
- Phase-based daily tasks with add/edit/delete and local completion tracking
- Study log that saves subjects, topics, hours, MCQs, and notes
- Topic-based MCQ generator with 100+ local questions, scoring, timer, explanations, history, and weak-topic tracking
- Free/legal content library with official/public source links and in-app reading cards
- Previous year paper section with public listings and clearly labeled practice papers
- Study materials filters, bookmarks, completion tracking, resource notes, and quiz generation
- Progress dashboard, mock tests, habits/journal, settings, export/import backup
- Offline-ready static PWA with relative paths for GitHub Pages

Deploy on GitHub Pages:
1. Push this folder to the repository main branch.
2. Enable GitHub Pages for the branch/root folder.
3. Open the Pages URL in Safari or Chrome.

Install on iPhone:
1. Open the GitHub Pages URL in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
4. Open CrackTrack from the home screen.

Notes:
- All personal data is stored in the browser's localStorage.
- Use Settings / Backup to export a JSON backup.
- External resources are stored as metadata and original summaries with source links.
- Practice papers are not labeled official unless the source is verified as official/public.
